## 1. 项目概述

### 1.1 项目背景

本项目针对第二十届全国大学生智能汽车竞赛-地瓜机器人智慧医疗挑战赛设计，旨在通过多传感器融合、路径规划、目标识别和大模型交互等技术，使智能车在模拟医疗环境中完成自主导航、目标识别、任务执行等任务。

### 1.2 项目目标

- 完成从起点P到任务发布点的自主导航
- 通过视觉识别二维码获取任务信息
- 根据任务要求（顺时针/逆时针）在黄色通道上运行
- 检测人形目标并通过大模型进行图生文描述
- 完成任务后自主返回起点P

### 1.3 主要功能

1. **二维码识别**：识别任务发布点的二维码，解析任务信息
2. **人形检测**：使用YOLOv8模型检测场景中的人形目标
3. **图生文**：通过大模型对检测到的人形进行描述
4. **路径规划**：根据任务信息规划顺时针或逆时针路径
5. **运动控制**：实现路径跟踪控制
6. **避障**：利用激光雷达实时检测并避开障碍物
7. **语音播报**：通过TTS播报任务信息和状态

## 3. 核心算法说明

### 3.1 二维码识别算法

#### 3.1.1 算法原理

使用pyzbar库进行二维码解码：
1. 接收摄像头图像
2. 将ROS图像消息转换为OpenCV格式
3. 使用pyzbar解码图像中的二维码
4. 解析二维码内容（顺时针/逆时针）
5. 发布任务信息

#### 3.1.2 关键参数

- 图像帧率：30 FPS
- 解码阈值：置信度 > 0.5
- 重复识别间隔：2秒（防止重复发布）

#### 3.1.3 代码文件

- `qr_detector_node.py`

### 3.2 人形检测算法

#### 3.2.1 算法原理

使用YOLOv8轻量级模型进行目标检测：
1. 加载预训练的YOLOv8n模型
2. 对输入图像进行目标检测
3. 筛选出person类别（ID=0）
4. 提取人形区域并裁剪
5. 将图像编码为base64发送给大模型

#### 3.2.2 关键参数

- 模型：YOLOv8n（nano版本，速度快）
- 检测阈值：confidence > 0.5
- 检测频率：每2秒1次（防止重复处理）

#### 3.2.3 代码文件

- `human_detector_node.py`
- `llm_interface_node.py`

### 3.3 图生文与大模型

#### 3.3.1 技术方案

支持两种大模型部署方式：
1. **云端大模型**：使用OpenAI GPT-4o API
2. **本地大模型**：使用本地部署的多模态大模型

#### 3.3.2 处理流程

1. 接收裁剪后的人形图像（base64编码）
2. 构建提示词，要求详细描述图像
3. 调用大模型API
4. 解析返回结果
5. 发布描述文本供语音播报

#### 3.3.3 提示词设计

```
请详细描述这张图片，重点关注：
1. 人物的数量和位置
2. 人物的姿态和动作
3. 周围环境
4. 其他重要特征
```

### 3.4 路径规划算法

#### 3.4.1 路径点定义

根据比赛场地定义关键路径点：
- P点：起点(0, 0)
- 任务发布点：(5, 2)
- 黄色通道入口：(5, 2.25)
- 诊疗区起点：(5, 2.5)
- 诊疗区终点：(5, 5)

#### 3.4.2 两种路径模式

**顺时针路径**：
```
P → 任务点 → 通道入口 → 环形起点 → (4.5,2.5) → (4.5,5) → (0.5,5) → (0.5,2.5) → 环形终点 → 通道入口 → P
```

**逆时针路径**：
```
P → 任务点 → 通道入口 → 环形起点 → (0.5,2.5) → (0.5,5) → (4.5,5) → (4.5,2.5) → 环形终点 → 通道入口 → P
```

#### 3.4.3 代码文件

- `path_planner_node.py`

### 3.5 运动控制算法

#### 3.5.1 控制策略

采用比例控制（P控制）：
1. 计算当前位置与目标路点的距离
2. 计算目标航向角
3. 线速度：与距离成正比，限制最大速度
4. 角速度：与角度误差成正比
5. 大角度误差时降低线速度，先转向

#### 3.5.2 控制参数

```python
# 目标位置容差（米）
target_position_tolerance = 0.3
# 最大线速度（米/秒）
max_speed = 0.5
# 最大角速度（弧度/秒）
max_angular_speed = 1.0
# 线速度比例系数
kp_linear = 0.5
# 角速度比例系数
kp_angular = 1.0
```

#### 3.5.3 代码文件

- `motion_controller_node.py`

### 3.6 避障算法

#### 3.6.1 避障原理

1. 接收激光雷达扫描数据
2. 找出最近障碍物距离
3. 如果小于安全阈值（0.5米），执行避障
4. 比较左右两侧平均距离，选择较空旷一侧转向

#### 3.6.2 避障逻辑

```python
if obstacle_detected:
    if left_avg > right_avg:
        turn_left()
    else:
        turn_right()
```

#### 3.6.3 代码文件

- `obstacle_avoidance_node.py`

2. 测试电机和舵机的运动方向
3. 验证传感器数据的正确性
4. 调整阿克曼转向参数

### 5.2 软件部署

1. 安装Ubuntu 22.04 LTS系统
2. 安装ROS2 Humble
3. 安装地瓜机器人相关软件包
4. 安装Python依赖：
   ```bash
   pip install opencv-python
   pip install pyzbar
   pip install ultralytics
   pip install requests
   pip install numpy
   ```

1. 创建ROS2工作空间
2. 将代码包复制到src目录
3. 编译工作空间：
